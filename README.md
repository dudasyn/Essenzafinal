# Essenza Sua Caza | Website Institucional

O repositório principal do website oficial da **Essenza Sua Caza**, onde apresentamos nossa visão, portfólio de projetos e serviços de construção, e informações de contato.

## Visão Geral

Este projeto é um site estático e responsivo, construído com foco em alta performance e facilidade de manutenção, servindo como a vitrine digital da construtora.

---

## Tecnologias Utilizadas

O projeto é baseado em um *stack* estático moderno e utiliza o seguinte conjunto de tecnologias:

* **Linguagens Core:** **HTML5**, **CSS3** e **JavaScript** puro (*Vanilla JS*).
* **Gerenciamento de Pacotes:** **npm** (para *scripts* de build, utilitários e dependências de desenvolvimento, se houver).
* **Hospedagem:** **Hostinger** (servidor compartilhado ou VPS, via File Manager ou FTP).
